# Contributing

1. Fork & clone.
2. `make install && pre-commit install`.
3. Create a feature branch: `git checkout -b feat/my-thing`.
4. `make test && make lint` before pushing.
5. Open a PR — CI must be green.

## Commit style
Use conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`.
